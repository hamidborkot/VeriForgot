# Paper Figures

This directory contains all figures used in the VeriForgot CRBL 2026 paper.

## Figure List

| File | Caption | Source |
|---|---|---|
| `fig1_architecture.png` | VeriForgot system architecture and verification pipeline | Generated by `src/` visualization script |
| `fig2_results.png` | MIA AUC per unlearning method (left) and model accuracy comparison (right) | Generated from `results/all_results.json` |
| `fig3_oracle.png` | Oracle detection results: genuine vs. fake unlearning AUC and confusion matrix | Generated by Kaggle Notebook 04 |

## How to Regenerate

```bash
python scripts/generate_figures.py
```

Or run **Kaggle Notebook 04** and download the output figures.

## Notes

- All figures use real experimental data from CIFAR-10 / ResNet-18 experiments
- `fig3_oracle.png` is generated at the end of the full experiment pipeline
- Figures are 180 DPI, suitable for both paper PDF and presentation slides
